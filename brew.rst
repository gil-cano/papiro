Instalación
-----------


instalar paquete:

    $ brew install <formula>


Listar paquetes::

    $ brew list


Actualizar lista de paquetes disponibles::

    $ brew update

listar paquetes no actualizados::

    $ brew outdated

actualizar paquete::

    $ brew upgrade <formula>
