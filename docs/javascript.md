# What sould I read to undertand React and JavaScript

To effectively understand React, a solid foundation in modern JavaScript (ES6+) is essential. Without it, grasping React's concepts and patterns will be significantly more challenging.

For JavaScript:

- [Eloquent JavaScript](https://eloquentjavascript.net/) by Marijn Haverbeke:
    This provides a comprehensive and in-depth understanding of JavaScript from the ground up, including advanced concepts.

- [You Don't Know JS](https://github.com/getify/You-Dont-Know-JS)
 series by Kyle Simpson:
    This series dives deep into the core mechanisms of JavaScript, revealing its nuances and often misunderstood aspects.

- Online resources like [MDN Web Docs](https://developer.mozilla.org/es/docs/Web/JavaScript) and FreeCodeCamp:
    These offer excellent, up-to-date documentation and interactive tutorials for learning JavaScript fundamentals and advanced features.

For React:

 -  [The Road to React](https://www.roadtoreact.com/) by Robin Wieruch:
    Highly recommended for beginners, this book focuses on practical application and building real-world projects.

- [Learning React: Modern Patterns for Developing React Apps](https://www.oreilly.com/library/view/learning-react-2nd/9781492051718/) by Alex Banks and Eve Porcello:
This book covers modern React patterns and best practices in detail.

- [Official React Documentation](https://react.dev/learn):
The official documentation is an invaluable resource for understanding React's core concepts, API, and best practices. It's constantly updated and provides clear explanations.


## Recommended Learning Path:

- Master JavaScript Fundamentals:
    Ensure a strong understanding of variables, data types, functions, control flow, arrays, objects, asynchronous JavaScript (Promises, Async/Await), and ES6+ features (arrow functions, destructuring, modules).
- Learn React Basics:
    Start with the official React documentation or a beginner-friendly book like "The Road to React" to understand components, props, state, and lifecycle methods (or Hooks).
- Practice Building Projects:
    Apply your knowledge by building small React applications. This hands-on experience is crucial for solidifying your understanding.
- Explore Advanced React Concepts:
    Once comfortable with the basics, delve into topics like Context API, Redux (for state management in larger applications), React Router, and performance optimization.
